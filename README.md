Velocity Runner: Neo Genesis — Rise of Bharat
Tagline: When the world falls, Bharat still runs.
A futuristic 3D endless runner for GitHub Pages. The game has a modern cyberpunk look with the core of Bharat: Neo Aryavarta, Surya Core, Surya Shards, Trinetra AI, and Maharakshak Titan boss events.
Main Identity
Character: Aarav Astra
World: Neo Aryavarta 2149
Core Object: Surya Core
Collectibles: Surya Shards
Enemy AI: Trinetra
Drone: Trinetra Drone
Boss: Maharakshak Titan
Controls
PC
Left / A: Move left
Right / D: Move right
Space / Up / W: Jump
Down / S: Slide
Shift: Dash
E: Slow motion
Mobile
Swipe left/right: Change lane
Swipe up: Jump
Swipe down: Slide
Tap quickly: Dash
Long press: Slow motion
Files
Upload these files to one GitHub repository:
```text
index.html
style.css
script.js
README.md
```
GitHub Pages Steps
Create a new GitHub repository.
Upload `index.html`, `style.css`, `script.js`, and `README.md`.
Go to Settings.
Open Pages.
Under Build and deployment, select Deploy from a branch.
Select branch: `main` and folder: `/root`.
Click Save.
Wait for GitHub Pages to generate your website link.
Note
This first version uses procedural 3D shapes, neon materials, bloom, fog, rain, city buildings, animated billboards, obstacles, collectibles, mobile controls, and local high score. Later versions can add real character models, sound effects, a shop, and more advanced boss attacks.
